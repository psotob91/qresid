# Validation evidence

`qresid` is a postestimation command. Validation therefore focuses on whether
the fitted conditional distribution is evaluated correctly, whether the
probability integral transform (PIT) uses the right support, and whether the
normal-score residual agrees with an independent calculation when one is
available.

This page gives the public validation summary for version 1.0.0. The installed
package remains small, while the GitHub repository keeps the routine smoke
tests and reproducible examples used to check package behavior.

The public release is checked under Stata 15.0 version mode with
`set varabbrev off`; see the [Stata version compatibility report](stata-version-compatibility.html).

## Summary by model class

| Model class | Stata command or specification | Validation approach | Independent reference |
|---|---|---|---|
| Gaussian continuous | `regress`; `glm, family(gaussian)` | Fitted normal CDF and normal-score replay | Base R `lm`/`glm` conventions and direct normal CDF calculations |
| Poisson counts | `poisson`; `glm, family(poisson)` | Discrete CDF endpoint replay with fixed uniforms | Base R GLM and Poisson CDF replay |
| Bernoulli and binomial | `logit`; `logistic`; `binreg`; `glm, family(binomial)` | Bernoulli/binomial endpoint replay with fixed uniforms | Base R GLM, binomial CDF replay, and Dunn-Smyth RQR formulas |
| Gamma | `glm, family(gamma)` | Gamma CDF replay, link checks, and `dispersion(#)` sensitivity replay | `statmod::qresiduals` conventions and direct Gamma CDF calculations |
| Inverse Gaussian | `glm, family(igaussian)` | Inverse-Gaussian CDF replay across documented links and `dispersion(#)` replay | `statmod` inverse-Gaussian probability calculations and direct CDF replay |
| Negative binomial | `nbreg`; documented NB variants | Mean and ancillary-parameter extraction, followed by NB CDF replay | `MASS` parameterization checks and manual NB CDF replay |
| Zero-inflated counts | `zip`; `zinb` | Mixture CDF replay for extra mass at zero plus the count component | Manual ZIP/ZINB CDF replay |
| Truncated counts | `tpoisson`; `ztp`; `tnbreg`; `ztnb` | Conditional CDF replay on the observable support | Manual truncated count CDF replay |
| Censored counts | `cpoisson` | Interval-PIT replay for left, right, and two-sided censoring | Manual censored Poisson CDF replay |
| Generalized Poisson | Stata Journal `gpoisson` | Generalized-Poisson CDF replay after the external estimator is installed | `VGAM::pgenpois0` and manual CDF replay |
| Hurdle counts | `hplogit`; `hnblogit` | Hurdle CDF replay: zero process plus positive truncated count component | `pscl::hurdle` and manual hurdle CDF replay |
| Frequency weights | Documented `fweight` specifications | Frequency-expansion checks; no final residual multiplication by weight | Expanded-data replay |
| Direct probability weights | Selected direct `[pweight=]` models | Model-based Stata diagnostic checks | No survey-design residual benchmark is claimed |
| Adjusted/studentized type | Selected unweighted `regress`/`glm` specifications | Leverage adjustment replay: `qres/sqrt(1-h)` | `glmtoolbox` Gaussian check and direct replay elsewhere |

## Public test entry points

The public repository keeps a compact test harness:

| File | Purpose |
|---|---|
| [`tests/run_all_tests.do`](../tests/run_all_tests.do) | Runs the routine Stata unit and smoke tests from a repository checkout. |
| [`tests/hardening_smoke.do`](../tests/hardening_smoke.do) | Checks package installation, help access, and example execution. |
| [`tests/check_support_report_consistency.R`](../tests/check_support_report_consistency.R) | Checks public documentation links and package metadata consistency. |
| [`examples/run_examples.do`](../examples/run_examples.do) | Runs the public example do-files. |
| [`tools/ci/run_stata_version_compatibility.ps1`](../tools/ci/run_stata_version_compatibility.ps1) | Runs the Stata 15.0 compatibility check with `set varabbrev off`. |

The workflow [`stata-tests.yml`](../.github/workflows/stata-tests.yml) runs the
Stata test harness on a self-hosted runner with Stata already installed.

## Datasets used in tutorials

| Dataset source | Used for | Type | Reproducibility link |
|---|---|---|---|
| Simulated Gaussian and GLM datasets | Continuous, binomial, count, zero-inflated, truncated, censored, weights, and dispersion examples | Generated by Stata scripts with fixed seeds | [manual asset script](scripts/build_manual_assets.do) |
| Stata `auto` | Gaussian and selected frequency-weight examples | Built-in Stata data | [Gaussian example](../examples/example_gaussian.do) |
| Stata built-in examples | Repository examples and quick-start diagnostics | Stata-distributed examples | [examples directory](../examples/) |
| Synthetic generalized-Poisson and hurdle datasets | Specialized count examples | Generated with fixed seeds | [manual asset script](scripts/build_manual_assets.do) |

Logs generated during local validation are written to ignored log directories.
They are not part of the installed package.
