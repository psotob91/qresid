# Project scope

`qresid` is not intended to be a generic residual plotting utility. The project
focuses on rigorous distribution-based diagnostics derived from fitted
probability models.

The current scope prioritizes generalized linear models, count-data
regression, discrete-response diagnostics, reproducible statistical
validation, and Stata/Mata implementations suitable for SSC distribution and
eventual Stata Journal submission.

Future versions are expected to incorporate multilevel extensions,
simulation-based residuals, correlated-data diagnostics, generalized additive
models, Bayesian-compatible residual diagnostics, and DHARMa-style simulation
workflows after the corresponding theory, extraction rules, and validation
evidence are documented.
